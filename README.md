## 주요 기능
| 카테고리           | 상세 기능                                                                                       |
| -------------- | ------------------------------------------------------------------------------------------- |
| **로그인 & 회원가입** | JWT Access/Refresh Token 인증<br>이메일 회원가입/로그인<br>Google/Kakao OAuth 로그인                       |
| **마이페이지**      | 작성 게시글/댓글 및 뱃지 확인<br>이메일/비밀번호/닉네임 변경<br>회원 탈퇴                                               |
| **알림**         | 게시글 좋아요/댓글 생성 시 알림<br>이벤트 기반 비동기 처리                                                         |
| **게시글 & 댓글**   | 게시글 작성/수정/삭제<br>Redis로 조회수 중복 조회 방지<br>게시글 좋아요(즐겨찾기)<br>댓글/대댓글 작성/수정/삭제<br>AI 댓글(OpenAI 기반) |
| **AI 상담**      | OpenAI 기반 상담/대화<br>Redis TTL(20분) 기반 상담 세션 관리                                               |
| **미션 & 배지**    | 루틴/활동/소셜/기타 카테고리 미션<br>경험치 지급, 배지 획득                                                        |
| **관리자 페이지**    | 사용자 및 게시글 관리<br>신고 관리<br>지원 사업 등록                                                           |
